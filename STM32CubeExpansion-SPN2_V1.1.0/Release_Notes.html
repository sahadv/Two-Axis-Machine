Release Notes for X-CUBE-SPN2

Software for STM32, Expansion for STM32Cube

Copyright 2015 STMicroelectronics

 


The X-CUBE-SPN2 software package is an expansion for STM32Cube. This software provides drivers running on STM32 for STMicroelectronics L6470, a fully integrated microstepping motor driver with motion engine and SPI. The X-CUBE-SPN2 is built on top of STM32Cube software technology that ease portability across different STM32 microcontrollers.

X-CUBE-SPN2 software features:
  • Read and write of the L6470 registers
  • Send the application commands to the L6470
  • Address any L6470 mounted in any stacked X-NUCLEO-IHM02A1 expansion board
  • Configuration of X-NUCLEO board (GPIOs, SPI, etc.)
  • Free user-friendly license terms.
  • Example implementation available for X-NUCLEO-IHM02A1 STM32 expansion board plugged on top of one NUCLEO-F401RE, NUCLEO-F302R8 or NUCLEO-F072RB.

The figure below shows the overall architecture.
     
         

  • At the bottom layer there are the HW components: the STM32 MCU and the L6470 DMOS driver for bipolar stepper motor.

  • The drivers abstract low level details of the hardware. The specific part for the X-CUBE-SPN2 are the BSP drivers for the L6470 configuration, the motor control and the X-NUCLEO-IHM02A1 expansion board setup.
  • The example show how to use the BSP drivers for configuration with one bipolar stepper motor based on a setup with a NUCLEO-F401RE or NUCLEO-F302R8 or NUCLEO-F072RB and an X-NUCLEO-IHM02A1 expansion board.
